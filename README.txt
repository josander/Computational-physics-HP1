Three C programs are provided:

alpotential.c : Contains functions that calculates the energy, forces, and 
                virial using an EAM potential for aluminum.


initfcc.c     : Contains a function that generates the positions of atoms in a 
                fcc lattice.

MD_main.c     : You should write your MD program in this file. This file also
                contains examples on how to use the functions in initfcc.c and 
                alpotential.c as well as an example on how to generate random 
                numbers.
